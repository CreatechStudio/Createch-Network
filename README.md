## Createch-Network是一个和谐友爱的服务器，所以您必须遵守以下规则

1. 在纯净生存（Pure Survival）当中，您不可以使用任何游戏外挂,连点器，来获取物资或杀戮其他玩家。

2. 您可以制作刷怪塔、制作空置域、等任何不超出游戏范围的机器，来构建“完善的后勤系统”。

3. 在服务器里面，禁止辱骂其他玩家。

4. 如果您认为您被恶意封锁了，应该立刻前往此仓库提交申诉请求（Issue），而不是在互联网上对于管理员实施人身攻击。

5. 如果您在Issue当中提出了很有建设性的意见 （您可以发送一个标签为“建议”的Issue，我们将尽快与您取得联系），我们将会有对应的奖励。

6. 应当尊重别人的点歌选择，但是同时您也不应该随意点一些奇怪的歌（如lost river）这会造成您失去点歌的权利。如果您与很多玩家都觉得这首歌不大合适，您可以使用/music vote来投票切歌，当票数大于一半时，会自动切歌。

7. 禁止在游戏内传播违法，宗教，色情内容，如果被举报、巡察发现，将会被永久ban-ip。

8. 请不要尝试去攻击服务器，我们保留发起反击的权利。

9. 如果您觉得自己是一个优秀的玩家，可以按照模板 https://github.com/CreatechStudio/Createch-Network/issues/13 发送一个标签为“内测申请”的Issue,提交内测申请，提前尝试最新玩法,我们将尽快与您取得联系。

10. ### 在您计划进行10秒内有大于等于10个TNT引爆的活动时，如制作空置域时，需至少提前一天 按照模板 https://github.com/CreatechStudio/Createch-Network/issues/21 发送一个标签为“爆破申请”的Issue，您必须在得到授权邮件后，方可开始作业。否则后果自负。

11. 祝您在服务器内度过美好的时光。

## 服务器内指令：


/music                                          
点歌插件

/music search 歌曲名                   
点歌

/music vote								  
投票切歌

/setwarp 名称
设立传送点，传送点所有人可见，如果设置错误，请联系管理员删除，如恶意设置节点，将被删除，恕不通知

/warp 名称                                   
一键传送(此指令可跨维度使用)

/sethome 名称
设置家，传送点仅自己可见

/home 名称
传送回家

/delhome 名称
删除家

/setspwan
设置出生点

/spawn
传送回出生点

/back
返回死亡地点

/tpa 玩家名                                   
请求传送至某玩家

/tpahere 玩家名                           
请求将某玩家传送至当前位置

/hat
将手上物品放在头盔位置

/editsign
更改告示牌内容，需要将指针指向告示牌

/rename
修改手上物品的名字，无需经验

如果遇到上述指令无法使用，请发送标签为BUG的Issue，或发送邮件到 service@createchstuio.com ，我们会尽快与您取得联系
