## UCS-Minecraft Server是一个和谐友爱的服务器，所以您必须遵守以下规则

1. 在纯净生存（Pure Survival）当中，您不可以使用任何游戏外挂，来获取物资或杀戮其他玩家。
2. 您可以制作刷怪塔、制作空置域、等任何不超出游戏范围的机器，来构建“完善的后勤系统”。
3. 在服务器里面，禁止辱骂其他玩家。
4. 如果您认为您被恶意封锁了，应该立刻前往此仓库提交申诉请求（Issue），而不是在互联网上对于管理员实施人身攻击。
5. 如果您在Issue当中提出了很有建设性的意见 （您可以发送一个标签为“建议”的Issue，我们将尽快与您取得联系），我们将会有对应的奖励。
6. 应当尊重别人的点歌选择，但是同时您也不应该随意点一些奇怪的歌（如lost river）这会造成您失去点歌的权利。如果您与很多玩家都觉得这首歌不大合适，您可以使用/music vote来投票切歌，当票数大于一半时，会自动切歌。
7. 禁止在游戏内传播违法，宗教，色情内容，如果被举报、巡察发现，将会被永久ban-ip。
8. 请不要尝试去攻击服务器，我们保留发起反击的权利。
9. 如果您觉得自己是一个优秀的玩家，可以按照模板 https://github.com/CreatechStudio/UCS-Server/issues/13 发送一个标签为“内测申请”的Issue,提交内测申请，提前尝试最新玩法,我们将尽快与您取得联系。
10. ### 在您计划进行10秒内有大于等于10个TNT引爆的活动时，如制作空置域时，需至少提前一天 按照模板 https://github.com/CreatechStudio/UCS-Server/issues/21 发送一个标签为“爆破申请”的Issue，您必须在得到授权邮件后，方可开始作业。否则后果自负。
11. 祝您在服务器内度过美好的时光。

## 服务器内指令：


/music                                          
点歌插件

/music search 歌曲名                   
点歌

/music vote								  
投票切歌

/tplist 地点                                   
一键传送(此指令仅限名称后维度使用，其他维度使用，后果自负)

/tpa 玩家名                                   
请求传送至某玩家

/tpa here 玩家名                           
请求将某玩家传送至当前位置
